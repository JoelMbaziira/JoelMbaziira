## The mental model I'm building

Reconnaissance is not about tools — it is about understanding how organisations build things. Every enterprise solves the same categories of problems: public-facing systems, remote access, developer environments, internal APIs, monitoring infrastructure. Each solution creates a predictable attack surface. The job is to read that surface before anyone else does.

**Current focus areas:**
- DNS internals (record types, zone walking, NSEC, DNSSEC)
- IP allocation hierarchy (IANA → RIR → ASN → subnet)
- HTTP security headers and what each one prevents
- TLS certificate analysis and certificate transparency logs
- Python tooling for automation
